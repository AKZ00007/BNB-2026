# How to Verify Your Token on BscScan

To get the green **"Verified"** checkmark on BscScan so the Honeypot Scanner gives you a perfect 100/100 score, follow these exact steps:

## 1. Get Your Flattened Source Code
I have generated the exact, flattened Solidity code for your token. It contains the main contract and all OpenZeppelin dependencies merged into one file.
- **Open this file in VS Code:** `z:\bnb-bangalore\bnb-project-1\flattened_token.sol`
- **Copy** all the text inside it (Ctrl+A, Ctrl+C).

## 2. Go to BscScan Testnet
1. Open your browser and go to: `https://testnet.bscscan.com/`
2. Search for your deployed token's contract address (e.g., `0x8fF...` from the scanner error).
3. Click the **"Contract"** tab with the little warning icon next to it.
4. Click the blue **"Verify and Publish"** link.

## 3. Fill in the Compiler Details
On the verification page, enter these exact settings (this is what your app used to deploy it):

- **Compiler Type:** `Solidity (Single part)`
- **Compiler Version:** `v0.8.20+commit.a1b79de6`
- **Open Source License Type:** `MIT License (MIT)`
- Click **Continue**.

## 4. Paste the Code and Submit
1. In the large "Optimization" dropdown, select **`Yes`**.
2. For "Runs", leave it at **`200`** (this is the default and matches your `hardhat.config.ts`).
3. Under **"Enter the Solidity Contract Code below"**, paste all the code you copied from `flattened_token.sol`.
4. Check the reCAPTCHA box at the bottom.
5. Click **"Verify and Publish"**.

## 5. Test the Scanner Again!
Once BscScan says "Successfully generated ByteCode and ABI for Contract Address", go back to your local app's Interactive Demo.

Hit the **Honeypot Scan** play button again. It will now query BscScan, see the green verified checkmark, successfully audit your tax logic, and give you a beautiful **100/100 LOW RISK** score! 🎉
